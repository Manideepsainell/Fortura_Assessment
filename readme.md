# Fortura Tech Assessment

## Overview

This repository contains solutions to 10 basic JavaScript programming problems focused on core concepts like loops, arrays, conditionals, and objects.

All solutions are implemented in a single file (`index.js`) with a controlled execution approach to run each problem individually.

---

## Questions Implemented

1. Even Counter – Count even numbers in an array
2. Sum of Prices – Calculate total from price list
3. Password Attempts – Allow 3 login attempts
4. Print Pattern – Generate star pattern using loops
5. Reverse Array – Reverse without built-in methods
6. Find Maximum – Find largest number in array
7. Shopping Cart – Apply discount and calculate total
8. Average Marks – Compute average from marks array
9. Frequency Counter – Count occurrences of elements
10. Order Billing System – Menu-based billing with GST

## How to Run

1. Install Node.js (if not installed)

2. Run the program:

```
node index.js
```

3. Open `index.js` and change the following line to run a specific question:

```js
const question = 1; // change from 1 to 10
```

---

## Notes

* Only one question runs at a time for clean output
* No built-in shortcuts used where restricted (e.g., reverse)
* Edge cases handled for most problems
* Readline is used only for password input (Q3)
